Static Container

This is a static content storage accessible via HTTP.

Any file added to static directory will appear here. For example the URL for current file is: http://localhost:8080/index.html.

If there is a file named example.png located in img directory, so its path is: img/example.png relative to the static root, it will be available at: http://localhost:8080/img/example.png

Note, that static directory content is excluded from the repo. You need to adjust .gitignore file to change this behavior.
